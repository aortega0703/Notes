For $(n>0)\in \mathbb{Z}$, Euler's totient function $\varphi(n)$ counts the number of positive integers up to $n$ relatively prime to it. That is$$
\varphi(n) = \bigg|\big\{k\in\mathbb{Z}:\ 0<k\leq n,\ \gcd(k,n)=1\big\}\bigg|
$$
