For a function $f:A\to B$
- $f$ is _injective_ if $f(a) = f(a') \implies a=a'$ for all $a\in A$.
- $f$ is _surjective_ if for all $b\in B$, $b=f(a)$ for some $a\in A$.
- $f$ is _bijective_ if it is injective and surjective.
